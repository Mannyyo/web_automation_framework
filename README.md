# web_automation_framework v0.1

Esqueleto inicial com:
- Browser wrapper (Selenium)
- Excel manager (pandas/openpyxl)
- Logger e decorators úteis
- Exemplo de uso

Como usar:
1. Instale dependências: pip install -r requirements.txt
2. Garanta driver do navegador no PATH (chromedriver/geckodriver) ou especifique driver_path
3. Rode exemplos em /examples
